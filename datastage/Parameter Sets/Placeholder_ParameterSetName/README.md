# Directory relating to a parameterset being managed by the PS_SPECIALHANDLING pipeline option

The name of this directory matches the name of the parameterset being managed, and the directory
contains a single parameter substitution file for each deployment environment.

Please refer to [this page](https://docs.mettleci.io/parammgmt) for a full explanation of how 
parametersets can be managed in this way, and the required contents of this directory.
